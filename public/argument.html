How to Create an Argument

By Oscar Walker-Hunter

What is an argument?

An argument a set of statements (called premises) that work together to support another statement (the conclusion).

How to create an arguement:

  1. Think of a premise. Such as "I am at the store"
  2. Think of a second premise that is inferred (using inference rules)
  3. Combine the premises to create the conclusion.
Ex:
Socrates is man.
All Men are Mortal.
Socrates is mortal
